# Windows Update Compliance Reporter

A read-only PowerShell toolkit for Windows update history and compliance reporting.

## Features

- Installed hotfix inventory
- Recent update history
- Windows Update service status
- Last successful update context
- CSV, JSON, and HTML reports

## Run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Windows_Update_Compliance_Reporter.ps1
```

## Safety

Read-only reporting only. No updates or services are changed.
